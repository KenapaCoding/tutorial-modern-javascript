
# Jawaban Pertanyaan Tentang Function Expression, Arrow Function, IIFE, Higher-Order Function, Callback, dan Recursion di JavaScript

## Jawaban Pilihan Ganda

1. **B) Function yang disimpan dalam variabel**
2. **C) Arrow function memiliki sintaks yang lebih singkat dan tidak memiliki `this` sendiri**
3. **C) Function yang dipanggil langsung setelah dideklarasikan**
4. **B) Function yang menerima function lain sebagai argumen atau mengembalikan function sebagai hasil**
5. **C) Function yang dipanggil sebagai argumen ke function lain**
6. **C) Function yang memanggil dirinya sendiri untuk menyelesaikan sebagian masalah**

## Jawaban Soal Tantangan

```javascript
const pangkat = (base, exponent) => {
    return base ** exponent;
};

// Memanggil function pangkat
let hasil = pangkat(2, 3);
console.log(hasil); // Output: 8
```
